Files Structure

After downloading and extracting template package you will find 3 folders:

  • HTML: ready to use HTML template;
  • ImbueGULP: if you want to totally customize Imbue, you can use this template;
  • Documentation: this help file.

This section describes HTML folder structure. More about ImbueGULP folder structure you can find out in ImbueGULP section.

Imbue is one page HTML template. So, mostly you will need one *.html file and supporting *.css, *.js, *.php files with some images. All the directories and files are well organized as it shown on the image bellow.

  • all HTML files are located in the root directory;
  • separate folders are created for CSS files, JavaScript files, fonts and images;
  • source-files folder contains all expanded CSS files, expanded JavaScript files, plugins and components files;
  • mail.php file for sending e-mails is located in the root directory;
  • .htaccess file is located in the root directory. This file contain rules for files caching on web server.

Upload to server

In order to start working with Imbue, all you have to do is copy chosen .html file along with supporting folders provided with this template on your FTP server.

CSS Files

All CSS files are placed in css folder and have to be included in the <head> tag. Imbue uses uncompressed CSS files. If you want to improve Google PageSpeed Insights performance, you can use minificated files from source-files → imbue-css-min-files .

<!-- Template Styles Start-->
<link rel="stylesheet" type="text/css" href="css/plugins.css">
<link rel="stylesheet" type="text/css" href="css/loaders/loader-*.css">
<link rel="stylesheet" type="text/css" href="css/main.css">
<!-- Template Styles End-->

CSS files that have to be included in chosen .html file:

  • plugins.css
  • loader-*.css
  • main.css

JavaScript Files

All JavaScript files are placed injs folder and have to be included before the </body> tag.

  • libs.min.js - all JavaScript libraries that are necessary for Imbue proper work, are сompiled in this file;
  • gallery-init.js - for photoswipe gallery initialization;
  • imbue-custom.js - includes other scripts calls and settings;
<!-- Load Scripts Start-->
<script type="text/javascript" src="js/libs.min.js"></script>
<script type="text/javascript" src="js/gallery-init.js"></script>
<script type="text/javascript" src="js/imbue-custom.js"></script>
<!-- Load Scripts End-->

* - loader color skin name (dark or light)

Icons

Imbue uses Ionicons 2.0.1 iconic font. You can use ionicons.com to easily find the icon you want to use. Icons are already compiled in the fonts.min.css file and ready for use.

To change icons, just change the class names: ion-ionic, ion-arrow-up, ion-arrow-resize.

        <i class="icon ion-social-facebook"></i>
    </a>
</li>

All available icons you can see on official website - ionicons.com.

Countdown Setup

Imbue template comes with two countdown types - full countdown and days only countdown.You can choose the one you like!

  • #countdown-large
  • #countdown

Find theLoad Scripts section at the end of<body> and make sure that all the necessary js files are added in the following order:

  • libs.min.js
  • imbue-custom.js
<!-- Load Scripts Start-->
<script type="text/javascript" src="js/libs.min.js"></script>
...
<script type="text/javascript" src="js/imbue-custom.js"></script>
...
<!-- Load Scripts End-->

Setting your own date for countdown script can be done inimbue-custom.js file injs folder:

$(function () {
    $('#countdown').countdown({until: $.countdown.UTCDate(+10, 2019, 11, 22), format: 'D'});
    $('#countdown-large').countdown({until: $.countdown.UTCDate(+10, 2019, 11, 22), format: 'DHMS'});
});

Visit KBW Countdown site for more details and settings -keith-wood.name/countdown.html

Contact Forms

Imbue template has integrated ajax contact form:

  • Write a Line Form

It's very easy to set up this form to receive feedback on your e-mail. First of all you need to make sure that mail.php file is added to the root directory of your project.

Then find <form> tag with the following id:

  • id="writealine-form"

in the *.html file. The form contains 3 hidden required inputs. Replace data in the value parameter of these hidden fields:

<form class="form form-dark contact-form" id="contact-form">
    <!-- Hidden Required Fields-->
    <input type="hidden" name="project_name" value="Imbue - Classy Coming Soon and Landing Page Template">
    <input type="hidden" name="admin_email" value="example@gmail.com"> <!-- Your E-mail Here-->
    <input type="hidden" name="form_subject" value="Write a Line Form Message">
    <!-- END Hidden Required Fields-->
    ...
</form>

JavaScript files that have to be included:

  • libs.min.js
  • imbue-custom.js
<!-- Load Scripts Start-->
<script type="text/javascript" src="js/libs.min.js"></script>
...
<script type="text/javascript" src="js/imbue-custom.js"></script>
...
<!-- Load Scripts End-->

Done! Now you can receive messages on your e-mail.

Follow these steps to get Mailchimp list URL:

  1. Login or create an account on mailchimp.com and go to the Audience section.
  2. Open All Audiences:
  3. Create a new Audience:
  4. Provide the required information for this audience:
  5. For this new Audience, go to Signup forms section:
  6. Select Embedded forms.
  7. Copy action URL from the form.
  8. Add https:// if it is not included with Mailchimp list URL.

Open imbue-custom.js file from js folder. Just replace our Mailchimp form action URL with your Mailchimp list URL:

$('.notify-form').ajaxChimp({
    callback: mailchimpCallback,
    // Replace this Mailchimp form action URL with your Mailchimp list URL
    url: 'https://besaba.us10.list-manage.com/subscribe/post?u=e8d650c0df90e716c22ae4778&id=18f865dbf3'
});

ImbueGULP

If you want to totally customize Imbue, you can use our all-inclusive ImbueGULP template. ImbueGULP includes Bootstrap 4 (grid), Gulp, Sass, Bourbon, Autoprefixer, Browsersync, Clean-CSS, CSS Beautify, Uglify, Bower and other packages for efficient and comfortable work.

ImbueGULP requires node, npm and git.

Getting started

Install Node Modules: npm i
Run ImbueGULP: gulp

Project structure

  • The source code is located in the app directory;
  • SASS files are located in the sass directory;
  • Bower packages installation path - bower_components;
  • The dist folder is a production folder. It contains ready project with optimized HTML, CSS, JS and images.

Gulp tasks

gulp: run default gulp task for web development;
build: build project to dist folder;
clearcache: clear all gulp cache.

If you have any questions about ImbueGULP, you can email us on support@mixdesign.club.
